# Visão geral

Para consumir o serviço do Panamah é necessário que o desenvolvedor siga os seguintes passos:

  • importar a biblioteca específica da linguagem de sua aplicação
    • delphi
    • java
    • javascript
    • python
  • chamar, no bootstrap da aplicação, a função init() da sdk
  • fazer o envio dos dados observando os modelos do esquema de dados e usando a função send()
  • caso precise chamar o envio do lote imediatamente, usar a função flush()
  • para acompanhar a montagem e o envio dos lotes, usar a função progress()

# Diagrama detalhado

Para os usuários da sdk todo o processo é transparente, estando visível apenas as funções básicas já apresentadas.
